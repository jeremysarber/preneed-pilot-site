# Preneed Pilot Website

Marketing website for Preneed Pilot, a pre-need software product for funeral home agents and teams.

## Project Type

Static HTML/CSS website.

## Files

Main pages include:
- `index.html`
- `how-it-works.html`
- `contact.html`
- `privacy.html`
- `terms.html`
- `thank-you.html`

Supporting files include:
- `styles.css`
- `sitemap.xml`
- `robots.txt`
- logo / SVG assets

## Deployment

This site is intended to be deployed with Cloudflare Pages.

No build step is required.

Recommended Cloudflare Pages settings:
- Framework preset: `None`
- Build command: leave blank
- Output directory: `/`

## Notes

The waitlist form connects to Mailchimp.
The contact form connects to Formspree.
Google Analytics is installed for basic site tracking.
